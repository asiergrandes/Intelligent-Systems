# Intelligent Systems Seminars

This repository contains the seminar projects developed during the **Intelligent Systems** course at the **University of Ljubljana** as part of the Erasmus programme.

The repository includes two different seminar projects focused on optimization algorithms and machine learning techniques applied to intelligent systems problems.

## Seminar 1 — Shortest Path Optimization using Genetic Algorithms

Development of a shortest path problem solution using **Genetic Algorithms** and evolutionary computation techniques.

### Main topics

* Graph optimization
* Genetic algorithms
* Evolutionary strategies
* Pathfinding problems
* Heuristic optimization

The objective of this seminar was to study how evolutionary approaches can be applied to complex optimization problems and compare their performance with traditional methods.

---

## Seminar 2 — Machine Learning Algorithm Comparison

Comparative analysis of several machine learning algorithms applied to classification problems.

### Algorithms analyzed

* Logistic Regression
* Support Vector Machines (SVM)
* Random Forest

### Main topics

* Supervised learning
* Model training and evaluation
* Classification metrics
* Performance comparison
* Data preprocessing

The goal of this seminar was to evaluate the strengths and weaknesses of different machine learning approaches under various datasets and conditions.
